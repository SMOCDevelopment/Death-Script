You can change settings directly in client.lua:

  enableRespawn = true,      -- Allow respawning at hospital
  enableRevive = true,       -- Allow reviving in place
  respawnDelay = 15,         -- Seconds before respawn is available
  reviveDelay = 10,          -- Seconds before revive is available
}

Installation Guide

  1. Download or clone this repository into your FiveM resources folder.
  2. Add ensure your_resource_name to your server.cfg.
  3. Make sure you have ox_lib installed and started before this resource.
  4. Configure client.lua to set your preferred revive and respawn timers.
  5. Start your server and test by dying in-game to see the custom death UI and features.
